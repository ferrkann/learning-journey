# Topics

This folder contains **topic pages**: cleaner, curated summaries of what I currently understand.  
Unlike `/notes` (chronological logs), pages here are updated over time as my understanding improves.

**TL;DR (ID):** `/topics` adalah rangkuman per topik (lebih rapi dan versi terbaik saat ini), sedangkan `/notes` adalah catatan proses belajar yang kronologis.

---

## How to use this folder

- Read a topic page for the **big picture**.
- Jump to `/notes` to see the **learning trail** and how ideas evolved.
- If I build something small (code/diagram/simulation), it lives in `/projects`.

---

## Page structure (typical)

Each topic page usually includes:
- **Scope** (what the page covers / doesn’t cover)
- **Core ideas** (my own explanation)
- **Key terms (EN)** with short Indonesian definitions
- **Mini examples** (optional)
- **Common confusions / pitfalls**
- **References** (books/papers/videos)

Language policy:
- Main explanation: **Indonesian**
- Titles + key terms: **English**

---
